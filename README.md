Getting Started
---------------

1. 获取token
  https://dev.yinxiang.com/doc/articles/dev_tokens.php
  点击“产品”对应的连接
  再点击相应的就可以了

2. 运行

```bash
python3 src/fuck_yinxiang ${YOUR_TOKEN}
```
